Metadata entry guidance

Columns that cannot be edited

If you downloaded a transfer metadata CSV template there are columns that contain generated metadata that are intrinsic to the records. These columns should therefore not be edited:

  • File path
  • Last date modified
  • Checksum

Columns that accept multiple values

If you downloaded a transfer metadata CSV template there are columns that contain generated metadata that are intrinsic to the records. These columns should therefore not be edited:

  • FOI exemption codes
  • Languages
  • Former reference

To enter multiple values use a comma (,) between each value, for example to enter three FOI exemption codes:

23, 27(2), 33
