Motivations and context

As an outcome of a workshop in June 2023 the team generated some How Might We statements that were categorised into a few separate areas. One of those was the ability of the user to resume an in-progress transfer and easily review all the metadata they have previously added.

Design changes

  • Changes to the user journey that introducing a new Summary List page for closure
  • This page is the start and end of the mini-journey taken by the user for every record they add metadata to.
  • Instead of presenting the Tree View as the first page it shows a table containing only the records with metadata and a summary of metadata added.
  • Descriptive metadata is addressed in version #66 with insights from this testing round
