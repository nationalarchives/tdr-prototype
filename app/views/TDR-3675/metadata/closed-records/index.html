Closed records metadata

What mandatory closed record metadata you must provide:

  • FOI schedule agreed date - the date the closure schedule was agreed by the Advisory Council and/or The National Archives
  • Closure start date - the date of the last change to this record
  • Closure period
  • FOI exemption code(s)

If a record contains sensitivities in the file name or description, you must provide the following information:

  • Alternative file name
  • Alternative description - You must provide a description in the 'Descriptive metadata' section before entering this

For more information about closed record metadata visit our FAQ (opens in a new tab).

If you have no closed records then click 'I have finished adding metadata'.
