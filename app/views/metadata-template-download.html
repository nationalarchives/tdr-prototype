Consignment: MOCK-TDR-1294

Download a transfer metadata CSV template

A CSV has been generated from your uploaded files. This can be used to prepare your consignment metadata for transfer. The CSV includes the following metadata columns with metadata extracted during file upload, which should not be edited:

  • File path
  • Last date modified
  • Checksum

It also includes the following empty columns for additional metadata:

  • FOI decision asserted date
  • Closure start date
  • FOI exemption codes
  • Closure period
  • Alternative title
  • Alternative description
  • Description
  • Date of the record
  • Language
  • Translated title of the record
  • Former reference
