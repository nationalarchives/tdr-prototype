Consignment: MOCK-TDR-1294

Download and complete your metadata template

To transfer your records using this service you will need to complete and upload a CSV.

The template contains a row for every uploaded file with prepopulated columns for specific metadata that was extracted from your records. It also contains some columns with pre-populated default values.

You will need to check the extracted metadata, check the pre-populated default values and provide as much additional metadata about the records as possible.
