Consignment: MOCK-TDR-1294

File upload restrictions

File size restrictions

There is a size limit of 2GB for an individual file and we ask that you upload no more than 500,000 files per consignment.

If you need to transfer larger files, contact us at tdr@nationalarchives.gov.uk.

Files and folders we cannot accept

We cannot accept files and folders which are password protected, zipped or contain slashes (/ and \) in the name. You must remove all thumbnail images (thumbs.db) and executable files (.exe). Empty folders will not be transferred.
