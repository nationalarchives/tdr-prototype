Guide to completing the metadata file

This guide provides a quick overview of how to complete the metadata file effectively. Below, you will find essential information on the metadata you need to fill in, ensuring that all necessary details are accurately captured. To assist you further, we have included a table that serves as a guide to filling in the spreadsheet. By following these instructions and using the table as a reference, you can efficiently complete the metadata file with ease. This is generic text do not consider.

Column title Details Format Requirement Example
filepath This is the file path/folder structure extracted upon upload, do not modify. Do not modify Mandatory
filename This is the file name extracted upon upload, do not modify. Do not modify Mandatory
date last modified This is the date last modified extracted upon upload, do not modify. Do not modify Mandatory
date of the record If the date last modified is not meaningful, please provide the date of the record. YYYY-MM-DD Optional 2020-03-01
description Provide a description which gives more context to the record. Text, max 8000 characters Recommended Draft business case for a Learning Resource Centre
closure status Closed records must have been approved by the Advisory Council. Only the words 'Open' or 'Closed' will be accepted. This is case sensitive and variations on these words will not be accepted. Open or Closed, default: Open Mandatory for closed record Closed
closure start date Closed record: Must match date of the record (if provided) else date last modified, must also align to date provided on closure application. YYYY-MM-DD Mandatory for closed record 2020-03-01
closure period Closed record: Provide a list of the number of years closure for each foi exemption code, in corresponding order Semi-colon separated list of numbers between 1 and 150 (with no spaces) Mandatory for closed record 40;80
foi exemption code Closed record: Provide the FOI exemption codes used to justify closure for the record, as approved by the Advisory Council This should be the same as the code(s) on the closure schedule for the record. Semi-colon separated list (with no spaces) Mandatory for closed record 27(1);33
foi schedule date Closed record: This date must match the one found on the closure schedule for the record. YYYY-MM-DD Mandatory for closed record 2020-03-01
is description closed Closed record: Yes means the description should also be closed. Yes or No, default: No Mandatory for closed record Yes
former reference Provide any reference which may have been used during the life of the record (it must not contain any line breaks). If you have multiple former references, these should be separated by semi-colons. Text, max 255 characters Optional CRDA/12
alternate description Closed record: If the description of this record is closed, please provide an alternate description to be displayed on Discovery while the record is closed. Text, max 8000 characters Mandatory if description is closed This is an alternative description
language Indicate if the record is in English, Welsh or both (English;Welsh). This is case sensitive and variations on these words will not be accepted. If you have records in other languages, please contact your digital transfer advisor Semi-colon separated list (with no spaces) default: English Mandatory English;Welsh
