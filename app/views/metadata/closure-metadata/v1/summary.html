What closure metadata you must provide for a closed record
  • FOI decision asserted - this is the date that the closure schedule was agreed by the Advisory Council and/or The National Archives.
  • Closure start date - this is the date of the last change to this record
  • Closure period
  • FOI exemption code(s)
What closure metadata you can optionally provide for a closed record

If the original title or description contains sensistive information you can provide alternatives, which will be visible to the public during the closure period:

  • Alternative title
  • Alternative description
